# JAMod 개발 가이드

## 소개
해당 라이브러리는 원래 혼자 사용할 목적으로 만들어졌습니다.

하지만 주변 분들께서 자신도 사용하고 싶다고 하셔서 가이드를 만들게 되었습니다.

그렇기 때문에 몇몇(서버와 관련된)기능은 아직 저만 사용할 수 있다는 점 양해 부탁드립니다.

> ⚠️️ **경고**
>
> 해당 가이드와 예시 코드는 제 모드의 코드를 학습시켜 AI를 통해 생성한 코드입니다. 
> 
> 일부 코드가 잘못 구현되어 있을 가능성이 있으니 예시 코드는 참고용으로만 사용해주세요!
> 
> 잘못된 부분이 있다면 꼭 얘기해주세요!
